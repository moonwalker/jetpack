# Jetpack

Webpack for Moonwalkers.

## Quick Start

Install:

```shell
$ yarn global add @moonwalker/jetpack
```

Create a new project:

```shell
$ jetpack create projectame
```

Working on the project:

```shell
$ cd projectame
```

Run dev server:

```shell
$ yarn start
```

Build for production:

```shell
$ yarn build
```
